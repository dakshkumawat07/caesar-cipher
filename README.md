# 🔑 Caesar Cipher Tool

A classical cryptography tool built as part of my cybersecurity learning journey.

## Features
- Encrypt text with a custom shift value
- Decrypt text with a known shift value
- Brute-force decryption (tries all 25 possible shifts)

## Usage
python3 caesar_cipher.py

## Concepts Learned
- Substitution cipher logic
- Why simple ciphers are cryptographically weak
- Foundation for understanding modern encryption (AES, RSA)
